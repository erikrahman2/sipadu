## 🎯 APA ITU PENGAJUAN PUBLIK?

Pengajuan Publik adalah fitur untuk masyarakat umum (bukan PNS/PA Staff) yang ingin membuat pengajuan perubahan data nikah.

**Siapa bisa pakai?**
- Pasangan suami-istri
- Salah satu dari mereka belum PNS
- Ingin perubahan data melalui portal online

**Bagaimana prosesnya?**
1. Isi form pengajuan online
2. Upload KTP Suami & Istri
3. Tunggu review dari tim PA
4. Terima hasil melalui WhatsApp

## ✏️ CARA MENGISI FORM

### 1️⃣ Data Suami

```
NIK Suami
  ├─ Harus 16 digit
  ├─ Contoh: 3171012611950029
  └─ Copy dari KTP

Nama Suami
  ├─ Harus sesuai KTP
  ├─ Contoh: Ahmad Riyanto
  └─ Maksimal 100 karakter

Alamat Suami
  ├─ Harus sesuai KTP
  ├─ Contoh: Jl. Sudirman No 123
  └─ Lengkap dengan nomor rumah

RT/RW
  ├─ Contoh: 05/12 (RT/RW)
  └─ Dari KTP bagian bawah

Kelurahan
  ├─ Harus sesuai KTP
  └─ Contoh: Cilandak

Kecamatan
  ├─ Harus sesuai KTP
  └─ Contoh: Cilandak
```

### 2️⃣ Data Istri

```
PENTING: Data Istri HARUS BERBEDA dari data Suami!

Khususnya:
  ⚠️  NIK Istri ≠ NIK Suami
  ⚠️  Nama Istri ≠ Nama Suami
  ⚠️  Data diambil dari KTP Istri (bukan milik Suami)
```

Isi sama seperti Data Suami (lihat di atas).

### 3️⃣ Nomor WhatsApp

```
Format yang diterima:
  ✅ 081234567890 (standar Indonesia)
  ✅ 0812 3456 7890 (dengan spasi)
  ✅ +6281234567890 (format internasional)
  ✅ 6281234567890 (dengan kode negara)

Contoh nomor valid:
  - 08282281394081 ✅
  - 081234567890 ✅
  - +601234567890 ✅

Catatan:
  • Nomor untuk menerima tracking WhatsApp
  • Pastikan nomor aktif & bisa terima WA
  • Pastikan bukan nomor salah!
```

### 4️⃣ Pilih Institusi

```
Dropdown: Pilih Institusi
  ├─ PA Cilandak
  ├─ PA Jakarta Selatan
  ├─ PA Bandung
  └─ ... (lihat dropdown)

Catatan:
  • Pilih sesuai lokasi Anda
  • Lihat nama institusi dari dropdown
  • Jangan ketik, pilih dari list!
```

### 5️⃣ Upload KTP

```
KTP Suami
  ├─ Format: JPG, PNG, atau PDF
  ├─ Ukuran: Maksimal 5 MB
  ├─ Contoh: scan KTP Suami
  └─ HARUS terisi (wajib)

KTP Istri
  ├─ Format: JPG, PNG, atau PDF
  ├─ Ukuran: Maksimal 5 MB
  ├─ Contoh: scan KTP Istri
  └─ HARUS terisi (wajib)

Cara upload:
  1. Klik "Pilih File"
  2. Pilih file dari komputer/HP
  3. Tunggu file ter-upload
  4. Nama file muncul di bawah
```

### 6️⃣ Setujui Persyaratan

```
☐ Saya setuju dengan persyaratan & ketentuan

HARUS di-CENTANG! 
✅ Klik checkbox
```

## ⚠️ HAL PENTING YANG HARUS DIKETAHUI

### Pembatasan Pengajuan

```
⚠️  HANYA BISA SUBMIT 3 KALI DALAM 15 HARI

Contoh:
  ❌ Pengajuan 1 (5 Jan): SUBMIT OK
  ❌ Pengajuan 2 (6 Jan): SUBMIT OK
  ❌ Pengajuan 3 (7 Jan): SUBMIT OK (INI YANG TERAKHIR)
  ❌ Pengajuan 4 (8 Jan): ❌ DITOLAK! 
     "Sudah mencapai batas 3 pengajuan dalam 15 hari"
     
  ✅ Pengajuan 4 (22 Jan): OK (15 hari setelah pengajuan 1)

Implication: JANGAN submit berkali-kali dalam waktu dekat!
```

### Validasi Data

```
❌ AKAN DITOLAK JIKA:

1. NIK tidak 16 digit
   • "NIK Suami harus 16 digit"

2. NIK Suami sama dengan NIK Istri
   • "NIK Suami dan Istri harus berbeda"

3. File tidak valid
   • "File hanya boleh JPG, PNG, atau PDF"
   • "File tidak boleh lebih dari 5 MB"

4. Nomor WA tidak valid
   • "Nomor WA harus 9-15 digit"

5. Data tidak lengkap
   • "Kolom [X] harus diisi"

6. Checkbox tidak dicentang
   • "Anda harus setuju dengan persyaratan"
```

### File Upload

```
✅ BISA UPLOAD:
  • Foto KTP (JPG/PNG)
  • Scan KTP (PDF)
  • Ukuran: Sampai 5 MB

❌ TIDAK BISA UPLOAD:
  • Format lain (BMP, GIF, DOCX, dll)
  • File > 5 MB
  • File corrupt/rusak
  • File kosong

💡 TIPS:
  • Scan dengan kualitas baik
  • Pastikan semua teks terlihat
  • Tidak boleh blur/gelap
  • Kedua sisi KTP (depan-belakang)
```

## 🆘 TROUBLESHOOTING

### Masalah: Form Tidak Load

```
Halaman kosong/error saat buka form

SOLUSI:
1. Refresh halaman (Ctrl+F5)
2. Buka incognito tab (Ctrl+Shift+N)
3. Coba browser lain (Chrome/Firefox)
4. Cek koneksi internet
5. Hubungi admin: support@sipadu.baperjuakta.go.id
```

### Masalah: Submit Button Tidak Berfungsi

```
Setelah klik "Kirim", tidak ada yang terjadi

SOLUSI:
1. Cek semua kolom sudah terisi
2. Cek checkbox sudah dicentang
3. Cek file sudah diupload
4. Refresh halaman & coba lagi
5. Cek browser console (F12 → Console)
   • Jika ada error merah, screenshot & kirim ke admin
```

### Masalah: "File Terlalu Besar"

```
Error: "File tidak boleh lebih dari 5 MB"

SOLUSI:
1. Cek ukuran file (klik kanan → Properties)
2. Jika > 5 MB, compress menggunakan:
   • Online: tinypng.com
   • Paint: Resize image
   • Photoshop: Export dengan quality lebih rendah
3. Upload file yang sudah dikecilkan
```

### Masalah: WhatsApp Tidak Terkirim

```
Sudah submit tapi tidak terima WA selama > 10 menit

SOLUSI:
1. Cek nomor WA di form (scroll ke atas)
2. Jika salah nomor:
   • Buat form baru dengan nomor benar
   • Jangan submit 2x form yg sama!
3. Pastikan WA aktif & bisa terima pesan grup
4. Jika masih tidak dapat:
   • Hubungi PA langsung
   • Bawa token nomor pengajuan
```

### Masalah: "NIK Sudah Pernah Submit"

```
Error: "Sudah mencapai batas 3 pengajuan"

SOLUSI:
1. Cek apakah sudah submit 3x dalam 15 hari
2. Tunggu 15 hari dari submit pertama
3. Atau gunakan NIK berbeda (istri sebagai yang utama)
4. Jika yakin belum pernah, hubungi admin
```
